---
uid: LoM.Super.Serialization
summary:
    <i>Included in the base framework</i><br>
    <br/>
    This namespace contains a set of classes that mock the Unity serialization system.<br/>
    They add the ability to handle properties in property drawing.<br/>
    If used with fields they will be serialized as normal, but properties can and will <b>not</b> be serialized.<br/>
    <br/>
    This namespace is used when creating custom drawers for the SuperBehaviours.
---
